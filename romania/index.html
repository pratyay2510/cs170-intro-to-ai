Romania traversal — BFS, DFS & UCS

Classic Romania road map from AIMA / course materials: each road is undirected in the model (both directions allowed in search); roads are drawn once on the map. Pick any start and goal city. Ties break alphabetically by city name. Edge labels are road distances.

What the algorithms optimize

  • BFS finds a path with the fewest edges (not necessarily minimum total distance).
  • DFS explores deeply first; the first complete path found depends on neighbor ordering.
  • UCS expands by minimum path cost so far — with nonnegative costs, the first time the goal is popped is optimal total distance.
